# NumatoGpio

Elixir module for communicating with [Numato 32-channel GPIO module](https://numato.com/docs/32-channel-usb-gpio-module-with-analog-inputs/).

def deps do

    {:numato_gpio, "~> 0.1.0"}
  ]
end
